# Maya Portfolio

This repository contains three Maya exercises demonstrating my current skills in animation, modeling, and rendering.

---

## Bouncing Ball

**File:** `maya_bouncing_ball.mp4`

Focus areas:
- Keyframe animation
- Camera setup and framing
- Lighting and rendering
- Timing and motion

This exercise was used to study basic physics, squash and stretch, and clean motion arcs.

---

## Walk Cycle

**File:** `maya_walk_cycle.mp4`

Focus areas:
- Character animation
- IK controls
- Weight shift and balance
- Pose transitions

A simple rigged character is animated to walk, demonstrating rhythm, body mechanics, and natural motion.

---

## Modeling Showreel

**File:** `maya_modeling_showreel.mp4`

Focus areas:
- Mesh modeling
- Form and shape exploration
- Viewport presentation

A short reel of different mesh objects modeled in Maya.

## Notes

All files are rendered from Maya. Each exercise represents foundational work and ongoing learning.
